# Reusing responses

This example shows how to use file system cache to share responses across calls.

## Demo

https://reuse-responses-at-build-time.vercel.app/

## How to Use

You can choose from one of the following two methods to use this repository:



### Clone

Execute [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app) with [pnpm](https://pnpm.io/installation) to bootstrap the example:

```bash
pnpm create next-app --example https://github.com/vercel/examples/tree/main/solutions/reuse-responses reuse-responses
```

Next, run Next.js in development mode:

```bash
pnpm dev
```

([Documentation](https://nextjs.org/docs/deployment)).
